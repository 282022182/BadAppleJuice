# BadAppleJuice
本项目参考自[EvilAppleJuice](https://github.com/ckcr4lyf/EvilAppleJuice-ESP32)项目，用于在安卓设备上发送蓝牙广播包，唤起(iPhone)设备的弹窗。

> [!NOTE]
> Issue里尽量提供问题相关截图，谢谢！

> 程序截图为v1.3版本，后续版本可能略有不同

- [x] 当前功能：
1. 自动发送广播包
1. 自定义弹窗设备（27种）
1. 发送随机设备广播包
1. 自定义广播功率，控制广播范围
1. 自定义广播间隔时间

- [ ] 未来功能：
1. 支持其他手机的弹窗（如果能做到的话）


- [ ] 当前问题：
1. 无法发送扫描包（低危害）
